# Email AI Agent - User Dashboard

这是一个基于 React + Vite + Tailwind CSS 构建的现代化邮件管理仪表盘。

## 主要功能

*   **四象限视图 (Priority Matrix)**：根据紧急程度和重要性自动分类邮件。
*   **全局主题系统**：支持 Light (浅色)、Nebula (深色星云)、Forest (深色森林) 三种全局主题，一键切换。
*   **个性化个人资料**：提供 Classic、Creative、Zen 三种不同的个人资料页面布局。
*   **邮件详情与操作**：AI 驱动的邮件摘要、自动回复建议和任务管理。

## 快速开始

1.  安装依赖：
    ```bash
    npm install
    ```

2.  启动开发服务器：
    ```bash
    npm run dev
    ```

3.  构建生产版本：
    ```bash
    npm run build
    ```
